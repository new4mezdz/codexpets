# Codex Pets

Unofficial community-made pets for the Codex desktop app.

Codex 桌面应用的非官方社区宠物合集。

## Xiaoxue / 小雪

A white-haired, blue-eyed schoolgirl pet with a long, random-like idle sequence, shy hover reactions, wall-hugging movement, thinking effects, and 16-direction pointer gaze.

白发蓝眸的校园少女桌宠，包含长周期随机感待机、害羞悬停反应、贴墙慢挪、思考问号和 16 方向鼠标注视。
